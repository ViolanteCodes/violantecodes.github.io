MariaDong.com

Full site design and build for author Maria Dong, including custom templating and auto-generating views for short stories and editorial testimonials.

iPhone 6
Desktop
iPad

Deployed to: MariaDong.com.

Client keywords/request included: "dark, beautiful, vibrant, starry, hopeful, futuristic."

Built on: Django, hosted on PA.

Special features include:

  • Custom CSS design from scratch, featuring animated header and main menu.
  • Fully responsive via flex-box and javascript mobile menu.
  • "Shutter-style" animated page transitions.
  • Integrated Django admin console for ease of database updates.
  • Custom short fiction app that auto-generates listings for three different views, including condensed, expanded, and filter by genre.
  • Custom testimonials app that auto-generates self-linking TOC, allowing for ease of navigation.
  • Custom blog app.
  • Contact form with SMTP integration and integrated captcha using Django-Simple-Captcha.
  • Two-click updates via Git pull and single-file database upload.
