MViolante.Com (Django-Bakery)

Django site leveraging class-based bakery buildable views. Bakes into static flat files for ease of porting to shared hosting.

View this project.

Built with: Python/Django, Django Bakery, Hosted on Shared Hosting.

Special features include:

  • Custom list/detail portfolio apps for websites and python projects.
  • An amount of flex-box usage that probably borders on unhealthy.
  • Custom CSS theme, "Leafy Dong", built from scratch.
  • Blazing fast load time due to leveraging static files and minimal styling (1 page css sheet!)
  • Easy to update, autopopulating mobile responsive gallery for portfolio projects.
